class Excel
<-- Esta clase almacena la informacion obtenida del archivo visto y luego se crean varios metodos dentro que ayudan a buscar una informacion en especifico:

header()--> este devuelve todas las caberas o Headers de la hoja de excel basandose desde content[0] donde cero representa la primera fila, que en el excel lo podemos ver como los nombres de cada columna.

rows()--> este devuelve una nueva coleccion de filas, incluyendole la funcion de slice que te permite decirle al sistema desde que punto partir, en este caso como ya el metodo header(), nos devuelve solamente la informacion de la primera fila, entonce no es necesario traer nuevamente esa fila, por ello empezamos a guardar la informacion desde la segunda fila que en este caso le colocamos "this.content.slice(1, this.content.length)"  en el primer parametro se envia el valor del cual queremos partir y en el segundo parametro le enviamos la cantidad de filas que encontro, es decir el tamaño total.

columns()--> este metodo es similar al metodo rows(), maneja la misma logica y funciones, solo que en este caso solo recorre las columnas y almacena esta informacion en ColumnsCollection.

class ColumnsCollection / RowsCollection --> estas dos clases almacenan la informacion recojida en los metodos rows() y columns(), respectivamente, en estas clases se agregan funcion mas especificas sobre los datos que de cada clase.

firtsCol() / firstRow() --> estas funciones se repiten en ambas colecciones de datos, pero devuelven datos especificos en cada caso, en el caso de "firtsCol" trae el primer dato de la columna asi mismo sucede con "firstRow" pero este devuelve el dato de la fila.

getCol(index) / getRow(index) --> estas funciones te devuelven un dato basandose en el indice que se le envio por parametro, claramente te devuelven datos de repestivas colecciones.

getRows() / getCols()--> estos metodos te vuelven toda la informacion respectivamente sin esperar un indice como lo hace el metodo anterior.

countRows() / countCols() --> estas funciones devuelven el tamaño/dimension  o cantidad de datos que se registro ya sea de filas o columnas.

class Row --> esta clase te permite acceder directamente y ejecutar funciones dentro de la fila, dentro de esta clase se crearon los siguientes metodos:

asesorNombre(excel) --> este metodo extrae y te retorna informacion obtenida previamente, este caso pedimos la informacion de todos los headers del excel y se almacena en "const header" usando "excel.header();" que nos devulve un array con todos los titulares, luego creamos "const index" al cual se le asignara el indice, este indice se logra hallar usando la funcion "indexOf()" asi como se ve se usa el array que anteriormente se creo header y se le ejecuta la funcion "header.indexOf('Valor a buscar')" esta funcion recibe como parametro el valor que se desea buscar, si lo encuentra este le devuelve el indice en el array.

Por consiguiente se crea un condicional para manejar la respuesta de la funcion, en el caso que el index sea diferente a -1, entonces retornara en el valor de la fila el index que se encontro, en caso contrario retornara un mensaje por consola.

Esta funcion nos ayuda a encontrar la columna donde se encuentra la informacion de los asesores/personal de la entidad, asi evitamos usar valores estaticos en el index y le pedimos al programa que lo busque por si solo.

NOTA:
Los siguientes metodos ejecutan la misma funcion dentro de ellas para encontrar el indice dinamicamente.

tiempoTotal(excel) --> este nos retorna donde se encuntra la columna que nos devuelve el tiempo total de atencion.

estado(excel) --> este metodo nos retorna la posicion del estado de la consulta si fue Finalizado o Abandonado.

tipoPersona(excel) --> retorna la posicion del tipo de cliente po ejempl; persona natural o persona juridica.
